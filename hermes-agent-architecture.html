Hermes Agent Architecture

v0.8.0

NousResearch's autonomous AI agent with multi-platform messaging gateway and skill-based tool system

Entry Points

  • • Hermes CLI — interactive shell with slash commands
  • • Gateway — Telegram, Discord, Slack, WhatsApp, Signal, QQBot, HomeAssistant
  • • ACP Adapter — VS Code, Zed, JetBrains IDEs
  • • Cron jobs — scheduled autonomous tasks

Agent Core

  • • AIAgent — chat loop, tool orchestration
  • • Prompt Builder — system prompt assembly
  • • Model Tools — LLM provider abstraction
  • • Context Compressor — auto-trim long context
  • • Prompt Caching — Anthropic cache control

Tool System

  • • Tool Registry — auto-discovers all tools/*
  • • 10+ core tools: terminal, file, web, browser, delegate, MCP, code_exec, skill_recommend, approval, cronjob
  • • 246 skills in ~/.hermes/skills/
  • • Self-improvement loop: recommend → log → analyze → patch → compound

Data & State

  • • HermesState/SessionDB — SQLite FTS5
  • • ~/.hermes/hermes_state.db
  • • Conversation persistence
  • • Full-text search across sessions
  • • skill_recommend + _usage_log.jsonl

Configuration

  • • ~/.hermes/config.yaml — model, tools, display, gateway
  • • ~/.hermes/.env — API keys (15+ providers)
  • • Provider: minimax (MiniMax-M2)
  • • Skin engine — CLI theming
  • • Profiles — multiple config sets

Development

  • • Branch: gateway-plugin-loading
  • • ~3000 tests in tests/
  • • Phased restart with drain (SIGUSR1)
  • • MCP client: 1050 lines
