# Key Concept: Skill Graph

A skill graph is a group of small Markdown files linked together using [[wikilinks]].

**Each file = one "knowledge node":**
- One job
- One rule
- One voice guide
- One workflow
- One audience profile
- One platform style

**The agent reads the graph, executes every step, and produces platform-native outputs.**
