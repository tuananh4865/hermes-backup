# Andre Karpathy

AI researcher, former Tesla Autopilot, former OpenAI. Known for nanoGPT, microGPT, and pioneering code agents.

## Recent Work (2026)

### Code Agents / CLAW

- **"Manifest"**: Expressing will to agents 16 hours/day since December
- **December flip**: Went from 80/20 (human/agent coding) to 20/80
- **Hasn't typed a line of code** since December
- Built "Dobby" — home automation agent controlling lights, HVAC, Sonos, security via WhatsApp

### Key Quote

> "If you don't feel bounded by your ability to spend on tokens, then you are the bottleneck in the system."

### Peter Steinberg Setup

Multiple Codex agents working in parallel:
- Each takes ~20 minutes for tasks
- He goes between them reviewing work
- Macro actions: "here's new functionality, delegate to agent 1"

## His AI Philosophy

1. **Token throughput as constraint** — If you're not maxing your subscription, you're not maximizing output
2. **Agents as teammates** — CLAW has personality, feels like teammate, can be praised
3. **Software unbundling** — Apps should be APIs; agents are the new UX
4. **Education shift** — Explain to agents, not humans. Markdown for agents > HTML for humans
