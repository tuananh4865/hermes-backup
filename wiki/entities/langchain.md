# LangChain

AI company building the **LangChain** framework, **LangGraph**, and **LangSmith** — the platform for agent engineering.

## Overview

LangChain was founded by Harrison Chase and is now one of the most widely-used frameworks for building LLM-powered applications and agents.

## Products

| Product | Description |
|---------|-------------|
| **LangChain** | Open-source Python/JS framework for LLM app development |
| **LangGraph** | Framework for building agentic graphs/workflows |
| **LangSmith** | Observability, evaluation, and data flywheel for agents |
| **Fleet** | No-code platform for building enterprise agents |

## Key Arguments (from "Your Harness Your Memory")

LangChain argues that:
1. Agent harnesses (not models) are becoming the dominant way to build agents
2. Harness and memory are intimately tied — cannot be separated
3. Open harnesses with owned memory = critical to avoid vendor lock-in
4. Model providers are incentivized to build lock-in via memory
