# OpenClaw

Open-source autonomous AI agent được tạo bởi Peter Steinberger (steipete). **Fastest-growing GitHub repo trong lịch sử.**

## Stats (Apr 2026)

- **Stars:** ~358K
- **Forks:** 72K
- **Contributors:** 360+
- **Peak growth:** 25,000 stars/24h — GitHub record
- **Timeline:** React 13 năm đạt 243K stars; OpenClaw đạt trong ~100 ngày

## Tech Stack

- **Agent runtime:** Pi (Mario Zechner's minimal harness)
- **Language:** TypeScript, Swift
- **License:** MIT

## Features

- Multi-platform messaging (WhatsApp, Slack, Discord, Telegram, Signal, etc.)
- Voice với ElevenLabs
- Browser control via Chrome DevTools Protocol
- 24/7 autonomous operation trên local hardware

## Why It Went Viral

1. Mobile-first UX — messaging interface thay vì CLI
2. Persistent always-on agent chạy local
3. Perfect timing — peak của "AI agent as personal assistant" narrative
4. MIT license, open-source
