# ChatGPT

> OpenAI's AI assistant built on GPT models. The dominant consumer AI product as of April 2026.

## Overview

**ChatGPT** is OpenAI's conversational AI assistant, launched in November 2022 and continuously updated since. It is the world's most widely used AI assistant, serving hundreds of millions of active users across consumer, education, and enterprise segments.

**Developer:** OpenAI
**First Released:** November 30, 2022
**Current Model Family:** GPT-5 series (GPT-5.3, GPT-5.4 as of April 2026)

## GPT Model Evolution in ChatGPT

| Model | Release | Key Features |
|-------|---------|-------------|
| GPT-3.5 | Nov 2022 | First broadly accessible LLM in chat form |
| GPT-4 | Mar 2023 | Multimodal (vision), improved reasoning |
| GPT-4o | May 2024 | Native audio, vision, text in single model |
| GPT-5 | Aug 2025 | Automatic reasoning, agentic capabilities |
| GPT-5.2 | Dec 2025 | Better code, spreadsheets, presentations |
| GPT-5.3/5.4 | Feb 2026 | Current default models (replaced GPT-5 Instant/Thinking) |

## GPT-5 Capabilities (August 2025)

GPT-5 was a landmark release that introduced **automatic reasoning** — the model determines when to apply Chain-of-Thought reasoning internally, without users needing to enable a separate mode.

Key capabilities:
- **Autonomous reasoning** — Applies Chain-of-Thought reasoning automatically when beneficial
- **High-quality code generation** — Produces production-ready code with minimal prompting
- **Front-end UI generation** — Generates front-end interfaces from natural language descriptions
- **Multimodal** — Text, images, audio, and document understanding
- **Agentic tools** — Built-in tool use, web browsing, and code execution

## GPT-5.2 (December 2025)

GPT-5.2 was designed with a focus on **economic value creation**:
- Better spreadsheet generation and manipulation
- Improved presentation building
- Enhanced code writing and editing
- Superior long-chain task execution

## GPT-5.3/5.4 (February 2026)

As of February 13, 2026, OpenAI retired GPT-5 Instant and GPT-5 Thinking. All ChatGPT conversations now default to GPT-5.3 (faster) or GPT-5.4 (more capable), selectable in model picker.

## Usage Statistics (2026)

- **200M+ weekly active users** (as of early 2026)
- **1B+ conversations per day** during peak periods
- Available in 140+ countries
- Supports 50+ languages

## Key Features in 2026

- **Voice Mode** — Real-time voice conversations with GPT-5
- **Canvas** — Collaborative document and code editing environment
- **Memory** — ChatGPT retains context across conversations
- **GPTs** — Custom versions of ChatGPT for specific use cases
- **Deep Research** — Agentic research capability that browses and synthesizes web content
- **Operator** — Browser agent that performs actions on behalf of users
