# Parakeet MLX

> A local speech recognition model optimized for Apple Silicon via MLX framework.

## Overview

Parakeet is a speech recognition model that runs efficiently on Apple Silicon through the MLX framework. MLX is Apple's machine learning framework specifically designed for Apple Silicon chips (M1, M2, M3, M4), enabling efficient local inference without cloud dependency.
