# Technical SEO

Technical SEO focuses on optimizing a website's infrastructure—site speed, crawlability, mobile-friendliness, and indexing—to improve search engine rankings.

## Key Areas

- Site speed and Core Web Vitals
- XML sitemaps and robots.txt
- Structured data and schema markup
- Mobile responsiveness
- HTTPS security
