# MLX Studio

> A tool or interface for working with MLX models on Apple Silicon.

## Overview

MLX Studio provides a user interface for running and experimenting with MLX models on Apple Silicon Macs. It enables developers and researchers to leverage Apple's unified memory architecture for efficient local LLM inference.
