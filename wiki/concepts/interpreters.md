# Interpreters

An interpreter executes code by reading source statements and executing them directly, line by line, without compiling to machine code first.

## Overview

Interpreters parse source code and execute it immediately, translating each statement into intermediate representations or direct execution. Languages like Python, Ruby, and JavaScript are traditionally interpreted.
