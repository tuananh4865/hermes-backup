# ARC (Automatic Reference Counting)

ARC (Automatic Reference Counting) is Apple's memory management mechanism for [[swift]] and Objective-C programs. Unlike garbage collection, ARC automatically inserts retain/release calls at compile time, freeing developers from manual memory management while maintaining predictable performance.
