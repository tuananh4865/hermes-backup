# Mistake Log

> **Purpose**: Global library of mistakes and lessons learned. Searchable across all projects.
> Every mistake is a learning opportunity. Log it, learn from it, prevent recurrence.

## Philosophy

1. **No blame culture** — Mistakes are data points, not failures
2. **Root cause focus** — Fix the system, not just the symptom
3. **Prevention over cure** — Easier to prevent than fix later

## How to Use

### Searching Mistakes
By tag:
```markdown
Search: tag:authentication
Search: tag:api
Search: tag:security
```

By severity:
```markdown
severity: critical
```

### When to Log a Mistake
- Bug took > 30 min to fix
- Same bug appeared before
- Review feedback caught an issue
- Production incident
