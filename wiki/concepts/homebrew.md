# Homebrew

Homebrew is the missing package manager for macOS (and Linux), enabling installation of software not included in the operating system.

## Overview

Homebrew simplifies installing development tools, runtimes, servers, and GUI applications through a command-line interface.

## Common Usage

```bash
# Install formulas
brew install node python git

# Install casks (macOS apps)
brew install --cask visual-studio-code slack

# Update packages
brew update && brew upgrade

# Search available packages
brew search <package>
```
