# LogSeq

LogSeq is an open-source outliner and knowledge management tool. It is local-first, markdown-based, and treats links as first-class citizens.

## Key Features

- **Local-first**: Data stored as plain markdown files
- **双向链接**: Bidirectional linking similar to [[Roam Research]]
- **Outliner**: Indented bullet structure
- **Daily Notes**: Journal entries on a calendar
- **Git-based**: Version control built-in via Git

## Comparison with Roam

| Feature | LogSeq | Roam |
|---------|--------|------|
| Cost | Free (self-host) | $15/mo |
| Data | Local markdown | Cloud-only |
| Open Source | Yes | No |
| Plugin Ecosystem | Growing | Mature |
