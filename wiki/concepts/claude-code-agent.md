# Claude Code — AI Coding Agent

Claude Code is Anthropic's CLI-based AI coding agent that competes with Cursor, GitHub Copilot, Windsurf, and Replit.

## Key Differentiators

- **Agent Skills** (anthropic/skills on GitHub) — teach Claude repeatable tasks
- **Native MCP integration** — uses Model Context Protocol for tool execution
- **Strong SWE-bench performance** — benchmark-leading on software engineering tasks
- **Unix philosophy** — CLI-first, composable with existing tools

## Skills System

Claude Code's skills system allows defining reusable task templates:
- Brand guideline adherence
- Code analysis patterns
- Document generation
- Custom tool orchestration
