# Programming Paradigms

Programming paradigms are fundamental approaches or styles of programming that shape how code is structured and organized.

## Common Paradigms

- **Imperative**: Step-by-step instructions modifying program state
- **Object-Oriented**: Organizing code around objects with properties and methods
- **Functional**: Treating computation as evaluation of mathematical functions
- **Declarative**: Expressing logic without describing control flow
