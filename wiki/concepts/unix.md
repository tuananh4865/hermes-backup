# Unix

Unix is a modular operating system philosophy built around simple, composable tools that do one thing well.

## Philosophy

- **Small tools**: Each command does one thing reliably
- **Text streams**: Tools communicate via plain text, enabling composition
- **Everything is a file**: Uniform interface for files, devices, and processes
- **Composability**: Pipe and combine tools for complex tasks

## Essential Commands

- `ls`, `cat`, `grep`, `sed`, `awk`
- `find`, `xargs`, `sort`, `uniq`
- `chmod`, `chown`, `ps`, `kill`
