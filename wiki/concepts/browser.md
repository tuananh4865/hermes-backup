# Browser

> Web browser automation and scraping — key capability for AI agents to interact with the web.

## Overview

Browser automation enables AI agents to interact with web pages programmatically. Key use cases:
- **Web scraping** — Extract data from websites
- **Form filling** — Automate web interactions
- **Screenshot capture** — Visual analysis of web pages
- **Web testing** — Automated UI testing

## Key Tools

| Tool | Language | Notes |
|------|----------|-------|
| Playwright | Python/JS | Maintained by Microsoft, used by Cursor |
| Puppeteer | JavaScript | Google's Chrome automation |
| Selenium | Multi | Legacy, cross-browser |
| browser-use | Python | AI agent native |
