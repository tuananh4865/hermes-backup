# Hermes Agent

**Hermes Agent** (by [[Nous Research]]) is an open-source, self-improving AI agent that runs persistently with memory across sessions. It builds reusable skills from experience and comes with 40+ built-in tools.

## Key Characteristics

- **Self-improving**: Creates skills from experience, improves them during use, nudges itself to persist knowledge
- **Persistent memory**: Maintains state across sessions without external databases
- **40+ built-in tools**: Native tool-calling for common agent tasks
- **Open source**: Built by Nous Research, available on GitHub
- **OpenRouter compatible**: Works with OpenRouter API

## v0.8.0 — April 8, 2026

The v0.8.0 release (April 8, 2026) is described as "the first open-source self-improving AI agent tutorial" — marking Hermes Agent as a reference implementation for self-improving agent architectures.
