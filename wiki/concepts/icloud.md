# iCloud

iCloud is Apple's cloud storage and computing service, providing seamless sync across Apple devices.

## Storage Tiers

- **iCloud+**: Paid plans starting at $0.99/mo for 50GB
- Includes: [[icloud]] Drive, Photos, Mail, Backup

## Key Features

- **iCloud Drive**: File storage accessible from any device
- **Photos Library**: Full-resolution photo sync
- **Keychain**: Encrypted password sync across devices
- **Find My**: Device location and tracking

## API Access

iCloud data can be accessed via:
- iCloud API (developer.apple.com/icloud)
- Third-party apps with iCloud permissions
