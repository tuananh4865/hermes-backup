# LLM Code Generation

LLM code generation uses large language models to produce, complete, or refactor source code based on natural language prompts or existing code context.

## Overview

Models like Claude Code, GPT-4, and specialized code models can understand programming intent and generate syntactically correct, functionally appropriate code across many languages and frameworks.

## Capabilities

- Code completion and suggestion
- Whole function or file generation
- Bug fixing and code review
- Test generation
- Code translation between languages
