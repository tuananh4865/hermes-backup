# Rapid MLX

**Rapid MLX** is an open-source local AI inference engine optimized for Apple Silicon, claiming to be the fastest local AI engine for Mac. It enables running local AI models without cloud dependencies or API costs.

## Overview

Rapid MLX addresses a key pain point for developers who want to run local AI models but need cloud alternatives when they become too expensive or slow. Built on Apple's MLX framework, it leverages Apple Silicon's unified memory architecture for efficient inference.

## Key Features

- **No cloud, no API costs** — runs entirely on Apple Silicon hardware
- **OpenAI-compatible API** — works with Cursor, Claude Code, and any OpenAI-compatible application
- **MLX-optimized** — leverages Apple's MLX framework for Apple Silicon
- **Multiple model support** — compatible with various MLX-model formats
