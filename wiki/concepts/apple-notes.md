# Apple Notes

Apple Notes is the native note-taking application developed by Apple for iOS, iPadOS, and macOS. It provides a free, integrated solution for capturing text, images, tables, sketches, and scanned documents with automatic iCloud synchronization across all user devices.

## Overview

Apple Notes has evolved from a simple text repository into a capable note-taking application with support for rich media, organizational folders, tagging with #tags, and collaboration features. It serves as the default notes app for millions of users who value seamless integration with the Apple ecosystem.

### Core Features

- **Folders and subfolders**: Hierarchical organization system
- **Tags**: Inline #tagging system for cross-folder categorization
- **Rich media**: Images, videos, scanned documents, sketches, and audio recordings
- **Lock notes**: Password or biometric protection for sensitive content
- **Collaboration**: Share notes via iCloud for real-time editing with others
- **Powerful search**: Full-text search including scanned document content

## Platform Availability

### iOS and iPadOS

Notes comes pre-installed on all iPhones and iPads. On iOS 16 and later, users can customize the toolbar, pin important notes, and use Lock Screen widgets for quick access.

### macOS

The macOS version offers additional features like:
- Quick Note from menu bar
- Continuity Camera support for inserting photos
- Full keyboard shortcuts
- Multiple window support

## Data Storage and Sync

Apple Notes stores data in iCloud by default, enabling seamless synchronization across devices. Notes are encrypted end-to-end when using iCloud, though the sync mechanism means data is stored on Apple's servers.

### Export Options

Users can export notes in multiple formats:
- Plain text
- PDF
- Individual images and attachments

## Comparison with Alternatives

Unlike dedicated note-taking apps such as [[obsidian]] or Notion, Apple Notes offers:
- **Pros**: Free, native integration, zero setup, automatic sync, privacy-focused
- **Cons**: Limited customization, no plugin system, less suitable for networked note-taking methodology

## Tips and Tricks

1. **Quick Capture**: Use Siri ("Hey Siri, take a note...") for hands-free capture
2. **Folders**: Long-press to reorder and nest folders
3. **Sketching**: Rotate and resize drawings with two-finger gestures
4. **Scan Documents**: Camera icon transforms physical documents into searchable text
