# Solo Dev AI — One-Person AI Development

> This page is a stub. The content below is a starting point — please expand with real knowledge.

## Overview

Solo dev AI refers to individual developers leveraging AI tools to build and ship products without traditional teams. In 2026, this became a recognized path to building billion-dollar companies.

## The One-Person Unicorn Phenomenon

Forbes: "The Future Is Solo: AI Is Creating Billion-Dollar One-Person Companies"

### Key Enablers

1. **AI coding agents:** Claude Code, Cursor, GitHub Copilot
2. **No-code/Low-code platforms:** Rapid prototyping without engineers
3. **AI agent workflows:** Automated marketing, support, operations
4. **Cloud infrastructure:** Serverless, managed databases

### Success Factors

- Speed: Ship MVPs in days not months
- Cost: Minimal infrastructure overhead
- AI leverage: 10x productivity per person

## Tools for Solo Devs

### Coding
- Claude Code, Cursor, GitHub Copilot
- vibe coding practices

### Infrastructure
- Vercel, Netlify (deployment)
- Supabase, Firebase (backend)
- Stripe (payments)

### Operations
- n8n, Zapier (automation)
- Intercom, Crisp (customer support)
