# SHA-256

SHA-256 is a cryptographic hash function producing a 256-bit hash value. It is widely used in blockchain consensus mechanisms, password hashing, and security protocols.

## Overview

Part of the SHA-2 family designed by the NSA, SHA-256 converts arbitrary input into a fixed-length digest that is computationally infeasible to reverse.
