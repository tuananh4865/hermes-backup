# CSS-in-JS

CSS-in-JS is a styling approach where CSS styles are defined within JavaScript code, enabling dynamic, component-scoped styling particularly popular in React ecosystems.

## Overview

Rather than separate `.css` files, styles are co-located with components in `.js` or `.tsx` files, allowing props and state to dynamically influence styling.

## Popular Libraries

- **Styled Components**: Template literal syntax for CSS
- **Emotion**: High-performance CSS-in-JS with flexible API
- **Chakra UI**: Component library with style props
- **Tailwind CSS**: Utility-first approach (not strictly CSS-in-JS but often paired)

## Benefits

- Scoped styles without specificity conflicts
- Dynamic styling based on props/state
- Easier code sharing and reuse
- Built-in vendor prefixing
