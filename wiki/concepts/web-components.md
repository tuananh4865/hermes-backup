# Web Components

Web Components are a set of web platform APIs enabling creation of reusable, encapsulated HTML elements with their own functionality.

## Technologies

- **Custom Elements**: Defining new HTML elements
- **Shadow DOM**: Encapsulated DOM and styling
- **HTML Templates**: Reusable template fragments
- **HTML Imports**: (deprecated) Importing HTML documents
