# Tailwind CSS

Tailwind CSS is a utility-first CSS framework that provides low-level building blocks (utility classes) for rapidly building custom designs without writing custom CSS.

## Overview

Unlike component-based frameworks, Tailwind provides single-purpose utility classes that can be composed directly in HTML markup.
