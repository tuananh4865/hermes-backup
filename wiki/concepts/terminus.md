# Terminus

> Minimal terminal coding harness — competitor to Pi Coding Agent on Terminal-Bench

## Overview

Terminus is a minimal terminal-based coding agent harness focused on CLI-first workflows. It competes with [[pi-coding-agent]] on the Terminal-Bench benchmark, both targeting developers who prefer terminal-based AI assistance over IDE-integrated solutions.

## Key Facts

- **Benchmark**: Terminal-Bench — evaluates coding agents on terminal-dependent tasks
- **Approach**: Minimal harness with direct shell access
- **Competitors**: [[pi-coding-agent]] (Pi), Claude Code, OpenCode
- **Differentiation**: Stripped-down, CLI-native design vs Pi's more structured approach

## Relationship to Pi Coding Agent

Pi Coding Agent and Terminus both compete in the minimal terminal coding harness space. Pi has gained more traction (42K stars, engine of [[openclaw]]), while Terminus remains a competitor on benchmark rankings.
