# autoresearch

> autoresearch is an open-source framework for autonomous feedback-driven research, open-sourced by Andrej Karpathy in March 2026.

## Overview

autoresearch enables AI agents to autonomously generate hypotheses, run experiments, evaluate results, and iteratively improve — without human-in-the-loop. It represents a concrete implementation of self-improving AI agent principles.

## Key Concepts

- **Autonomous Feedback Loops**: The agent generates its own training signal from experimental results
- **Hypothesis-Driven Research**: Agents form hypotheses and design experiments to test them
- **Iterative Refinement**: Each iteration improves the agent's performance on the target task
