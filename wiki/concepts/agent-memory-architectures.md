# Agent Memory Architectures

Critical distinction: **RAG ≠ Agent Memory**. Per arXiv:2602.02007, while RAG targets large, heterogeneous corpora with diverse passages, agent memory is a **bounded, coherent dialogue stream** with highly correlated context.

## Why This Distinction Matters

| Aspect | RAG | Agent Memory |
|--------|-----|-------------|
| Data | Large, diverse corpus | Bounded conversation stream |
| Retrieval | Keyword/semantic matching | Contextual continuity |
| Use case | Knowledge base Q&A | Ongoing task memory |

## Memory Solutions

- **Mem0** — Self-improving memory for AI agents
- **Zep** — Production-ready agent memory
- **Letta** — LLM-powered querying of memory
- **Cognee** — Memory infrastructure for LLMs
