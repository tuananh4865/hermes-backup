## NHỮNG CÔNG THỨC CỐT LÕI NHẤT

### Storytelling 4 Bước
```
1. Giới thiệu nhân vật chính (tính cách, quê quán)
2. Nêu vấn đề đám đông đang gặp
3. Trình bày giai đoạn khó khăn + cách vượt qua
4. Rủi ro / hi sinh để đạt kết quả
```

### 3 Chất Liệu Câu Chuyện
```
1. Đảo ngược tình huống → đồng cảm
2. Lịch sử ngành nghề → uy tín
3. Quay quy trình sản xuất → tin tưởng
```

### Nguyên Lý Viral
```
Chủ đề người ta TÌM + Phong cách RIÊNG = VIRAL
→ Mỗi ngành đều có chủ đề "7/10 người thành công"
```

### Nguyên Tắc Vàng
```
"Quay được hơn quay đẹp. Hoàn thành hơn hoàn hảo."
"Không có ngành nào không bán được. Chẳng qua chưa biết cách."
"Học chùa mãi rồi. Từ 2026 là trả đi."
```
