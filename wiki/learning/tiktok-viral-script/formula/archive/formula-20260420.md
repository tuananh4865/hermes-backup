## KỸ THUẬT DẪN DẮT MỚI

### Honest Negative Review Technique (Trinh)
```
1. Test nhiều vị trí: trán → tay → nách (tăng credibility)
2. Soften negative: "hơi vô dụng" không phải "vô dụng"
3. Balance: "cũng vui đó" trước khi phủ định
4. Final score: kết thúc bằng điểm số hoặc rating
```
→ Học được: Không cần phải all-positive để viral. Honest negative với demo cụ thể cũng viral.

### Scarcity/Exclusive Hook (Panda)
```
1. Giải thích backstory: tại sao bạn có sản phẩm trước
2. "Bán lén" / "độc quyền" language
3. "Fan cuồng" confession để tạo relatable
4. Promise để reveal thông số
```
→ Học được: Exclusive access narrative tạo intrigue mạnh hơn con số.

### Scoring System (Panda)
```
Mỗi sản phẩm = điểm cụ thể:
- DJI Flip: 7 điểm
- Action 6: 5.75 điểm
- Instax: 3 điểm

→ Dễ nhớ, dễ so sánh, audience có thể debate
```
→ Học được: Scoring = memorable takeaway.
