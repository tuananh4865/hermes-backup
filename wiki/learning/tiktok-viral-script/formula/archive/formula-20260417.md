### Empathy Hook vs Drama Hook
```
Empathy (Trinh): "Muốn X nhưng ngại Y" → relate → solve
Drama (Puta): "Không làm X = coi như chưa Y" → FOMO → action
→ Chọn hook type tùy niche:
  - Skincare/Beauty → Empathy
  - Food → Drama/Situation
  - News/Knowledge → Claim phản trực giác
```
