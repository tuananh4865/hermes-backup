## Chuyển VTT sang text

```python
import re

def parse_vtt(path):
    with open(path) as f:
        content = f.read()
    lines = content.split('\n')
    result = []
    for line in lines:
        line = line.strip()
        if not line or line.startswith('WEBVTT'):
            continue
        if '-->' in line:
            continue
        line = re.sub(r'<[^>]+>', '', line)
        result.append(line)
    return ' '.join(result)
```
