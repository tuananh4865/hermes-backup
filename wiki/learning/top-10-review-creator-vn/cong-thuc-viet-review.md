# Công Thức Viết Review — Áp Dụng Chung

**Nguồn:** Phân tích Puta Ăn Gì + Trinh Meow + Duy Thẩm học từ @duymuoi

---

## CÔNG THỨC 1: HOOK — 3 LOẠI

### A. Câu Táo Bạo (Puta style)

```
Dùng cho: Đồ ăn, trải nghiệm, địa điểm
Công thức: "Không [hành động] ở [địa điểm] = coi như chưa [trải nghiệm]"
Ví dụ:
- Không uống cà phê muối ở Huế = coi như chưa đến Huế
- Không ăn bún bò ở quán này = coi như chưa ăn bún bò Huế
```

### B. Câu Hỏi Tự Kiểm Tra (Trinh style)

```
Dùng cho: Sản phẩm, skincare, tech
Công thức: "Bạn có biết...? / Thử làm xem..."
Ví dụ:
- Mọi người thử vuốt hết tóc ra sau xem có lộ da đầu không
- Bạn có đang dùng sai cách sản phẩm này không?
```

### C. Câu Sốc (Duy Thẩm / DuyMuoi style)

```
Dùng cho: Kinh doanh, tài chính, xu hướng
Công thức: Nói con số đi ngược số đông
Ví dụ:
- 38 cách mở đầu video bán hàng — video bán khẩu trang
- 12 dự đoán xu hướng TikTok 2026
```

## CÔNG THỨC 2: CẤU TRÚC BODY

### A. Review Đồ Ăn (Puta style)

```
1. Tìm thấy quán (internet / bạn bè giới thiệu)
2. Đến quán → mô tả nhanh không gian, vị trí
3. Gọi món → gọi gì, giá bao nhiêu
4. Ăn thử từng món → cảm nhận từng món (ngon / không ngon)
5. So sánh → với quán khác, với kỳ vọng
6. Kết → recommend hay không, cho ai
```

### B. Review Sản Phẩm (Trinh style)

```
1. Intro: Đây là [sản phẩm], Trinh review vì [lý do]
2. Lưu Ý: 2-3 điều quan trọng TRƯỚC KHI dùng
3. Thành Phần: Số cụ thể, so sánh với sản phẩm phổ biến
4. Công Dụng: Chia theo loại da / đối tượng
5. Cách Dùng: Hướng dẫn từng bước
6. Kết Luận: Tóm tắt, recommend cho ai
```

### C. Review Tech / Kinh Doanh (Duy Thẩm style)

```
1. Vấn đề: Đang gặp vấn đề gì?
2. Giải pháp: Sản phẩm / cách này giải quyết thế nào?
3. Con số: Đo lường được (thu nhập, tăng trưởng, hiệu quả)
4. Kinh nghiệm: Chia sẻ cá nhân — thành công + thất bại
5. Kết: Recommend / không / tùy hoàn cảnh
```

## CÔNG THỨC 3: TỪ VỰNG

### Từ cảm xúc (dùng vừa phải)

```
TRÍCH NHỰNG: "trời ơi", "mê nhất", "quá chỉ sợ"
LÀM MỀM:     "nha mọi người", "nhá", "ấy"
NHẤN MẠNH:   "xứng đáng", "quốc dân", "nhất nhì"
```

### Từ chuyên ngành (LUÔN giải thích ngay)

```
KHÔNG NÓI: "SPF 50+ PA ++++, UVA, ceramide"
MÀ NÓI:    "SPF 50+ (chỉ số chống nắng cao nhất)
            PA ++++ (chống tia UVA rất tốt)
            ceramide (chất dưỡng ẩm tự nhiên trong da)"
```

### Từ so sánh (dùng nhiều)

```
"Béo hơn quán X"
"Không ngon bằng nhưng mà vẫn ô kê"
"Giống như bạn đang skincare cho da mặt vậy"
```

## CÔNG THỨC 4: NHỊP ĐIỆU

### Nhịp cho video NGẮN (< 30 giâi)

```
Nhịp đều suốt — không thay đổi
Mỗi câu 2-4 giây
Không có khoảng trống
```

### Nhịp cho video DÀI (> 60 giâi)

```
NHANH:  Mô tả hành động, sự kiện
CHẬM:   Cảm nhận, cảm xúc
NHANH:  Kết luận, so sánh
→ Xen nhuyễn theo cảm xúc thật
```

## CÔNG THỨC 5: KẾT — CALL TO ACTION

### Kết cho review TÍCH CỰC

```
"Mọi người mà biết quán nào ngon nữa thì
recommend giúp cô ta với nhá bái bai"

"Tóm lại em này xứng đáng là một [tên sản phẩm] quốc dân"
```

### Kết cho review TRUNG LẬP

```
"Không quá ngon cũng không quá dở
Tùy vào khẩu vị mỗi người nha"
```

### Kết cho review PHỦ ĐỊNH

```
"Mình không recommend vì [lý do cụ thể]
Nhưng nếu bạn thích [đặc điểm đó] thì có thể thử"
```
